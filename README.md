##Iniciando projeto

###Pré requisitos
* JAVA 8 
* Maven
* Ide Eclipse

###Instalação
* Download das dependências: Botão direito no projeto -> Maven -> Update Project

###Iniciando Projeto
* Run Configurations -> Arguments -> Program arguments: add "--spring.profiles.active=local"
